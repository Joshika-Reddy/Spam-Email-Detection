# Spam-Email-Detection
📌 Overview

This project classifies emails as Spam or Not Spam (Ham) using the Naive Bayes algorithm.

🧠 Algorithm

Naive Bayes is a probabilistic classifier based on Bayes’ Theorem, widely used for text classification.

⚙️ Steps
Data preprocessing
Feature extraction (Bag of Words / TF-IDF)
Train Naive Bayes model
Predict spam or ham
🛠️ Technologies
Python
Scikit-learn
Pandas
📊 Output
Classifies messages as:
Spam (1)
Ham (0)
